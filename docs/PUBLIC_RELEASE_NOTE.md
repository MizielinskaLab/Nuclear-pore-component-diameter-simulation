# Public release note

This folder is organized as a public-facing representative release for the Fig. 4-linked computational workflow.

Internal review-oriented notes were removed from this package. The repository now keeps only public-facing documentation needed to understand:
- what part of the manuscript this code supports;
- what software and external files are required;
- what order the workflow should be followed in; and
- what the current reproducibility scope is.


See `LICENSE_NOTE.md` for the public-use terms applied to the author-generated code and representative example data in this release.
