# Code availability statement

This repository provides the **author-generated MATLAB workflow**, representative inputs, and representative outputs for the spot-analysis and simulation-comparison component of the manuscript.

It is most directly linked to the **Fig. 4 computational workflow**, including:
- two-colour spot-analysis from microscopy-derived inputs;
- random-position simulation analysis; and
- comparison between experimental and simulated distributions.

This public release is a **representative worked example package**. It does not yet include the full paper-level simulation search space or every image-analysis input used across the study.
